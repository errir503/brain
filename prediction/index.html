Modèle de Prédiction

Cette application a pour but de visualiser les sorties d'un modèle de prédiction généré par un réseau de neurones spécifique : un auto-encoder.

Le modèle de prédiction reçoit les 25 canaux d'entrée du casque connecté. Par son modèle de réseau de neurones, il ressort ces données traitées par 5 neurones de sortie, chacun représentés par une couleur différente.

La représentation actuelle affiche les neurones de sortie au cours du temps, selon un ordre décroissant de poids sur les résultats.
